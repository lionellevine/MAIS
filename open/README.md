# Open problems

Short expositions of single open problems. Each opens with its AI safety motivation, then states the problem as precise mathematics — or reaches toward precision; a problem need not know its final statement yet. Problems here get incorporated into [research agendas](../agendas/).

Open problems are numbered MAIS-O1, MAIS-O2, … in order of arrival; numbers are permanent, so *MAIS-O1* always refers to the same problem. Problems posed inside the [research agendas](../agendas/) are numbered too: [INDEX.md](INDEX.md) locates every open problem in the repository.
