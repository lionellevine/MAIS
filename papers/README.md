# Papers

Papers in progress, public before they are polished — typically spun off from a [research agenda](../agendas/). Substantial contributors are invited as coauthors; see [CONTRIBUTING.md](../CONTRIBUTING.md).

Papers are numbered P1, P2, … in order of arrival; numbers are permanent.

| # | Paper | Source | PDF | Status |

| P1 | *Math for AI Safety: An Invitation for Mathematicians* | [TeX](P1/MAIS-P1.tex) | [PDF](P1/MAIS-P1.pdf) | under review · revision in progress |

Each paper lives in its own subfolder with its LaTeX source and compiled PDF. To rebuild one, run `latexmk -pdf -interaction=nonstopmode -halt-on-error FILE.tex` from its folder; LaTeX auxiliary files are ignored.
